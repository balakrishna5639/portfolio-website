# 🚀 Balakrishna Avaru - Interactive Portfolio Website

A stunning, super interactive portfolio website with modern design, smooth animations, and engaging user experience.

## ✨ Features

### 🎨 Design
- **Modern Dark Theme** with vibrant gradients and glassmorphism effects
- **Animated Background** with floating gradient orbs
- **Particle System** with connecting lines for visual depth
- **Responsive Design** that works on all devices
- **Premium Typography** using Inter and Outfit fonts

### 🎭 Interactive Elements
- **3D Card Tilt Effect** on hover for skill cards and project cards
- **Magnetic Button Effect** that follows mouse movement
- **Smooth Parallax Scrolling** for depth and dimension
- **Animated Skill Progress Bars** that fill on scroll
- **Typing Effect** for the hero name
- **Counter Animations** for statistics
- **Scroll Progress Indicator** at the top of the page
- **Back to Top Button** with smooth scroll

### 🎯 Sections

1. **Hero Section**
   - Animated greeting and name
   - Professional title with gradient text
   - Call-to-action buttons
   - Social media links
   - Animated avatar with floating tech icons

2. **About Section**
   - Professional summary
   - Statistics cards (Internships, Projects, Certifications)
   - Code snippet display

3. **Skills Section**
   - Frontend Development (JavaScript, React, HTML/CSS, Tailwind)

   - Backend & Database (Node.js, Express, MongoDB, SQL)
   - AI & LLMs (LangChain, Prompt Engineering, AI Agents)
   - Tools & Technologies tags

4. **Experience Section**
   - Timeline layout with animated dots
   - Walkingtree Technologies (July 2025 - December 2025)
   - Surya Tech Technology (June 2024 - July 2024)
   - Tech stack badges

5. **Projects Section**
   - DevConnect (Developer social networking platform)

6. **Education & Certifications**
   - B.Tech in Data Science & AI from ICFAI

### 🎪 Special Features

- **Dynamic Gradient** that changes based on scroll position
